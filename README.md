# MERN JWT MUI

## To Run this MERN App : 

- Server folder : 

```bash
  cd server 
  npm install
```

- Client folder : 

```bash
  cd client
  npm install 
```

- Main Folder : 

```bash
  npm install 
  npm run dev
```

### Details :

- Frontend : React 19
- Backend : Express.js
- Database : MongoDb 
- UI : Material UI
